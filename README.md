**# aLine Studio

aLine Studio is a browser-based creative production suite originally built for object shows, now expanding into animation, scripting, music, and storytelling workflows.

It combines timeline-based animation tools, scripting systems, and structured production features into a unified workspace.

---

## 🚀 Features

- 🎬 Timeline-based animation system
- 🧅 Onion skin support
- 🧱 Layer-based editing
- 📦 Custom project format (.aln ecosystem)
- 🎭 Bracket / elimination-style systems for storytelling
- 📝 Script + notes integration
- 🎧 Experimental audio/music workflow tools
- 💾 Auto-save (IndexedDB/local storage)

---

## 🧠 Core Idea

aLine is designed for:
- Object shows
- OC/AU storytelling projects
- Indie animation workflows
- Experimental browser-based production systems

It focuses on **structured storytelling through timelines, scripts, and interactive production tools**.

---

## 📁 Project Structure

- `Studio.tsx` – main editor workspace
- `App.tsx` – app routing & core layout
- `main.tsx` – entry point
- `DeviceModeModal.tsx` – mobile/desktop selection
- `TitleScreen.tsx` – startup screen
- `settings/` – preferences & configuration system

---

## 💾 File Format

aLine uses a custom project format:

- `.aln` → main project file (animation + structure)
- future expansions may include `.alna`, `.alns`, `.alng`
